Bebidas San Martin - Demo en Render

Esta carpeta ya esta preparada para subir a Render.

Comandos/configuracion:
- Build Command: pip install -r requirements.txt
- Start Command: gunicorn app:app
- Plan: Free

Usuarios de demo:
- selene / 1234
- vale / 12345

Importante:
Render Free sirve para mostrar el sistema, pero no es recomendable como sistema definitivo
usando SQLite. En el plan gratis, los cambios guardados en bebidas.db pueden perderse
cuando Render reinicia, redeploya o apaga la app por inactividad.

Dos opciones para ofrecer:
1. Demo gratis en Render:
   Link publico para mostrar el sistema. Ideal para probar, vender o presentar.
   No recomendado para datos reales.

2. Version seria:
   Hosting pago o servidor propio, con base persistente y backups.
   Ideal para uso diario del negocio.

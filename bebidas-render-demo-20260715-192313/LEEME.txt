Bebidas San Martin - version online

Para abrir el sistema:
1. Hacer doble clic en iniciar_web.bat
2. Entrar desde el navegador a http://127.0.0.1:5000

Usuarios cargados en la copia web:
- selene / 1234
- vale / 12345

Esta version usa la base web_app\bebidas.db, que es una copia de la base original.
La app de escritorio no fue modificada.

Para entrar desde otra computadora o celular en la misma red:
1. Dejar esta PC encendida con iniciar_web.bat abierto.
2. Buscar la IP de esta PC.
3. Entrar desde el otro equipo a http://IP-DE-ESTA-PC:5000

Para publicarlo en internet con dominio o link publico hace falta subirlo a un hosting
con servidor Python y configurar seguridad, backup y acceso a la base.
